# Journal

A small Journal app for Android, Third Year University Project

Project finished on March 1st 2026

## Features

- Create journal entries with title, content, date, and tag
- Journal entries are saved in local database
- Journal entries can be viewed, edited or deleted
- Journal entries can be filtered by title, content, date or tag
- Day or Night mode with [Catppuccin](https://catppuccin.com/palette) Latte or Mocha colorschemes
- User Interface text in English or French based on system language

## Development

Developed in Android Studio with Android API 36.0
